🔄 Sales Order Dynamic Approval

Multi-level approval workflow for quotations

This module enables configurable approval chains for Sale Orders based on Sales Teams.

  • ✅ Role-based approval steps
  • 🔔 Auto-notifications to next approver
  • 📊 Visual status: "Pending Approval: [User]"
  • 🔄 Auto-sync approvers from Team → Quotation

🎯 User Workflow

  1. Sales User: Create quotation → Click Confirm → System validates approval rights
  2. System: If approved, notify next approver via bell + email
  3. Sales Manager: Review → Approve/Cancel → Order proceeds to warehouse

⚙️ Configuration

1. Go to Sales → Configuration → Sales Teams

2. Open a team → "Sale Order Approver(s)" tab

3. Add approvers in sequence order (drag to reorder)

4. Assign this team to quotations to activate workflow

💡 Tip: Sequence 10 = first approver, 20 = second, etc.
