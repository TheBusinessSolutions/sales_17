Select Show Product Name or Description
=======================================

Installation
------------
1. Copy the module folder to your addons path.
2. Update the Apps List in Odoo.
3. Install 'Select Show Product Name or Description'.

Configuration
-------------
1. Go to **Sales** > **Configuration** > **Settings**.
2. Scroll down to the **Quotations & Orders** section.
3. Locate the **Report Content** setting.
4. Check or uncheck **Print Product Name** and **Print Product Description**.
5. Click **Save**.

Usage
-----
* Create a Quotation and add products.
* Print the Quotation (PDF).
* The 'Description' column will update based on your settings.
