# omg_sales_realtime_dashboard

Advanced Odoo 17 sales dashboard in Odoo Man style.

## Main capabilities
- Executive KPI cards with previous-period comparison
- Searchable multi-filter dropdowns with presets and favorites
- Dashboard mode and grouped list mode
- Drill-down from charts to sale orders
- Targets by team or salesperson
- Approximate profitability analytics
- Alerts, pivot summary, CSV export, widget personalization
- Static UI with all animations removed
